CODING ENVIRONMENT SYSTEM

Coding Environment System is a desktop and online application for hosting contests, trainings and events for programmers. Our goal is to provide a platform for programmers to compete and have fun. Coding Environment System is a noncommercial organization operated by CES team.

This help guide will give you the description on how to use the software.

SUBMISSION

To submit a solution to any question, follow these steps

  1. Select the question to which you want to submit the solution

  2. Select the language type for program to be submitted

  3. Select the file from "Select File" button

  4. Submit the program and wait for output

CONTESTS

TEST CONTESTS

These type of contests are used to conduct tests. Here, student are required to know the question code to attempt any question.

PRACTICE CONTESTS

These type of contests are used for practice. Here, students are free to solve any question without knowing the code.
